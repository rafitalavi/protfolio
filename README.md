Portfolio Starter (Django + React)
=================================

What's included:
- frontend/ (Vite + React + Tailwind)
  - Dark mode toggle implemented. Tailwind configured for 'class' dark mode.
- backend/ (minimal placeholder)

How to run frontend locally:
1. cd frontend
2. npm install
3. npx tailwindcss init -p   # if not already initialized
4. npm run dev

Notes:
- The frontend expects a Django backend at http://localhost:8000 for projects/contact endpoints.
- The CV placeholder PDF is at public/cv/YourName_CV.pdf
